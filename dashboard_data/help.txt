дешборд предоставляет возможности:
-мониторить, кто и когда сколько файлов отправил на сервер (+ различные фильтры)
-смотреть логи главного сервера в real-time
-обавлять/удалять/менять данные для входа юзеров в трекер

Тут есть разделение на роли:
0-сапорт, 1-модератор, 2-админ

Сапорт может только просматривать данные юзеров, не видит логи сервера, не может менять и удалять юзеров
трекера. Может добавлять новых юзеров

Модератор может редактировать юзеров трекера, не может их удалять, не видит логи.

Админ имеет полный доступ ко всем функциям дешборда

Эти данные находятся в БД в users -> role. Реализована защита как на стороне клиента, так
и на стороне сервера (каждый запрос проходит проверку прав юзера, который его отправил)
